Lariat Python SDK
=============================================
Lariat Data (`www.lariatdata.com <www.lariatdata.com>`_) is a Continuous Data Quality Monitoring Platform to ensure data products don't break even as business logic, input data and infrastructure change.


The Python SDK allows you to interact with the Lariat API and perform various operations like querying indicators, fetching datasets, etc. via code.
